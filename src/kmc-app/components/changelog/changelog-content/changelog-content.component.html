Beta Version 4.4.0 (05/02/2018)
The Beta KMC version is the first version of the new and improved Kaltura Management Console (KMC), Kaltura’s central media management application that provides Admins one intuitive interface for all their media management and publishing needs. The new KMC is HTML5 based and Flash free. It includes additional features and functionality, better performance, and has been redesigned to provide a consistent and engaging user experience across the entire platform.
 
New Functionality
 
Content Creation and Editing
  • Large file upload: Upload support for videos greater than 2GB
  • Upload monitoring: presents status on all current uploads. This feature is located in the top navigation bar, for quick access from any page on the KMC 
  • Editing: Access to the Kaltura Video Editing Tools including Clip and Trim are now available in the KMC. 
  • Advertisement module: The Kaltura Video Editing Tools now include advertisement management functionality. 
 
Entries List & Details
  • YouTube entries: YouTube entries are now visible in the entries list. 
  • Player view and entry details: Each entry page now includes the media preview as well as the entry details. These details are also available when navigating between tabs. 
 
Share & Embed
  • Sorting through players: Sorting through the list of players is now available, providing the ability to sort by name, upload date, and last date updated, making it easier to locate specific players. (Image)
  • Note: The player drop down list has been updated and all outdated players have been removed
 
Distribution 
  • Dashboard: Asset level dashboard of all active distribution connectors.
  • Link to 3rdparty site: A link to the distributed entry on the 3rd party site (YouTube, Facebook etc.) is available post export.
  • Entry remove button: Easy remove button that provides the option to delete the exported asset from the 3rd party site.
 
Retired Functionality
  • KMC Dashboard: The KMC dashboard has been removed and users logging into the KMC will now first see the Content page. Account and Billing information, previously housed on the bottom right corner of the KMC dashboard, can be now found in the Account Information page.
  • Notification Settings: Previously available under Integration Settings. All notifications set ups are now available via the new event notification module in the Admin Console and the email notification module in Kaltura MediaSpace (KMS).
  • Record from Webcam: Note that this functionality is still available in Kaltura MediaSpace via Kaltura’s capture tools.
 
Functionality NOT available in Beta
Note that all the following functionalities are still available in the legacy KMC, should you need to use them:
  • Replace Video
  • Thumbnail cropping
  • “Forgot password”
  • Syndication to iTunes
  • Instruction screens for syndication channels
  • Support for small resolutions (1024x768)
  • Foreign language support: Note that only English is supported at this time.
  • Entry flavors: Ability to link flavors to remote storage and match media files to drop folders
  • “Help”: Please refer to the User Guide document provided with your introduction email.
  • Analytics: Kaltura Analytics is currently undergoing a major overhaul. The new Analytics, slated for release in Q3 2018, will replace the current Flash-based offering and as a result Analytics has not been included in this Beta.
 
Known Limitations
  • Table Columns: Table columns size currently cannot be modified.
  • Trimming: Cases in which ‘approval before replacing’ is required, approvals must be made within the legacy KMC to be fully supported.
  • Navigation breadcrumbs: Note that back arrow on entry details pages will refer users to the entries list page, rather than the last page viewed.
  • Upload Monitoring: The bulk upload status is updated every 10 seconds, which means that status changes will be reflected at this rate.
  • Login: Users that open the same KMC in two tabs within the same browser will need to login into both pages.
